<h2 id="description">üìö Description</h2>

Things List is a backend application developed to manage tasks in a practical and efficient way, following the structure of a basic CRUD (Create, Read, Update, Delete). Inspired by the productivity concept of tools such as Todoist, this API provides full support for creating, updating, listing and deleting tasks.

<h2 id="installation">‚öôÔ∏è Installation</h2>

- 1: Clone this repository: `git clone https://github.com/victorozoterio/things-list-backend.git`;
- 2: Create a `.env` file from the `.env.example` file;
- 3: Fill in all the necessary variables in the `.env`;
- 4: Install the dependencies, running the command: `npm install`;
- 5: Upload the docker container, running the command: `docker compose up -d`;
- 6: Run the application, running the command: `npm run start:dev`
